We have a added a new feature using which you can swap any tokens you have in your wallet to SOL. <br/><br/>

### Steps to follow to sell the tokens :
    **Step 1** : Use the `/wallet` command to go to your wallet.
    <img src="../../../wallet.png" alt="A wild meteor" width="500" height="500"></img>
    **Step 2** : Select `Manage wallet`. <br/>
    **Step 3** : Select the `Sell Tokens` button.
<img src="../../../sell-token.png" alt="A wild meteor" width="500" height="500"></img>
    **Steo 4** : Select the % of amount you want to sell (Eb : 100%.50%).<br/>
    <img src="../../../select-percentage.png" alt="A wild meteor" width="500" height="500"></img>
    **Step 5** : Select the token from the list. <br/>
    <img src="../../../select-token.png" alt="A wild meteor" width="500" height="500"></img>
    **Step 6** : The selected tokens will be swapped back to `SOL`. ( Add sell-success image).
<img src="../../../sell-success.png" alt="A wild meteor" width="500" height="500"></img>

- You can use this feature in cases where you collect fees in tokens and later on decides to swap it to SOL.
