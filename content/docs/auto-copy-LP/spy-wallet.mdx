MetEngine supports LP notifications, alerting you whenever tracked wallets take LP positions. You can track up to 5 wallets simultaneously!

## Setting Up Wallet Tracking

### Step 1: Start the Bot
1. Open Telegram and go to [@met_engine_bot](https://t.me/met_engine_bot)
2. Type `/start` and press enter
3. Click on the `Auto Trade` button
<img src="/wallet.png" alt="Main menu" width="500" height="500"></img>

### Step 2: Add Wallets
1. From the Auto Copy Trade menu, click the `Add Wallet` button
<img src="/cp2.png" alt="Track Wallet" width="500" height="500"></img>

2. Enter wallet information using format: **`WalletName WalletAddress`**  
Example: `AlphaWallet 5F3sa2TJ...`
<img src="/cp3.png" alt="Add wallet" width="500" height="500"></img>

3. Repeat to add up to 5 wallets

### Step 3: Set to Notification Mode
1. To receive notifications without copying trades:
   - Tap the wallet to toggle to ðŸ”” mode
   - In ðŸ”” mode, you'll only receive notifications
   
2. To both receive notifications AND copy trades:
   - Tap the wallet name to toggle to âœ… mode
<img src="/sw1.png" alt="Tracked wallets" width="500" height="500"></img>

## What You'll Receive in Notifications

Each notification includes:

- **Wallet Name** 
- **Pool Name** with a link to Meteora
- **Token Amount** with the value in USD
- **Fees Claimed**
- **Strategy type**
- Links to **Dexscreener** and **GmGn**

## Managing Tracked Wallets

### Removing Wallets
To stop receiving notifications for a wallet:
1. Click the `Delete Wallet` button
<img src="/cp8.png" alt="Delete Wallet" width="500" height="500"></img>
2. You'll see a numbered list of your tracked wallets
<img src="/cp9.png" alt="Delete Wallet" width="500" height="500"></img>
3. Input the number of the wallet you want to remove and press enter
<img src="/cp10.png" alt="Delete Wallet" width="500" height="500"></img>
4. Wallet deleted successfully
<img src="/cp11.png" alt="Delete Wallet" width="500" height="500"></img>
