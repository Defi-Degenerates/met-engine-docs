## Setting Up Auto Copy Trading

### Step 1: Start the Bot
1. Open Telegram and go to [@met_engine_bot](https://t.me/met_engine_bot)
2. Type `/start` and press enter
3. Click on the `Auto Trade` button to begin
<img src="/wallet.png" alt="Main Menu" width="500" height="500"></img>

### Step 2: Add Wallets
1. From the Auto Copy Trade menu, click the `Add Wallet` button
<img src="/cp2.png" alt="Auto Copy Trade" width="500" height="500"></img>

2. Enter wallet information using format: **`WalletName WalletAddress`**  
Example: `AlphaWallet 5F3sa2TJ...`
<img src="/cp3.png" alt="Add wallet" width="500" height="500"></img>

3. Repeat to add up to 5 wallets

Note: When you add a wallet, Auto Copy is enabled by default (âœ…).

### Step 3: Configure Copy Settings
1. Click the `Configure` button next to any wallet
<img src="/cp5.png" alt="Configure button" width="500" height="500"></img>

2. Choose from these configuration options:
<img src="/cp6.png" alt="Configure menu" width="500" height="500"></img>

   - **Fixed Amount**: Always use the same SOL amount for every position
     - Example: Set 0.5 SOL â†’ Every trade uses exactly 0.5 SOL
   
   - **Portfolio %**: Use a percentage of your current SOL balance
     - Example: Set 10% â†’ If you have 5 SOL, trade with 0.5 SOL per position
   
   - **Copy Trader Position %**: Copy a percentage of the target's trade amount
     - Example: Target trades 1 SOL â†’ Set 50% â†’ You trade 0.5 SOL
     - This is amount-based: percentage of their actual trade size
   
   - **Same Proportion**: Mirror the same proportion of target's portfolio allocation
     - Example: Target trades 10 SOL from their 100 SOL wallet (10% of portfolio) â†’ You automatically trade 10% of YOUR portfolio
   
   - **Same Amount**: Copy the exact same SOL amount as the target trader
     - Example: Target trades 1 SOL â†’ You also trade 1 SOL

3. For trader-dependent options (Copy Trader Position %, Same Proportion, Same Amount), you can set a **Max Amount** to limit position size:
    - Example: If you set Max Amount to 2 SOL and the trader opens a 10 SOL position:
    - With "Same Amount" selected, you'd only use 2 SOL instead of 10 SOL
    - With "Copy Trader Position %" at 50%, you'd use 2 SOL instead of 5 SOL
    - With "Same Proportion", you'd be capped at 2 SOL regardless of the calculated proportion

## Managing Wallets

### Adding More Wallets
To add more wallets (up to 5 total):
1. Follow [Step 2: Add Wallets](#step-2-add-wallets) above

### Removing Wallets
1. Click the `Delete Wallet` button
2. You'll see a numbered list of your tracked wallets
3. Input the number of the wallet you want to remove and press enter
4. Wallet deleted successfully
