### Encountered a new token in the trenches and want to turn it into a money printing opportunity ? 
**We got you covered, MetEngine now supports creating Dynamic AMM pools on Meteora in less than 10 seconds.**

**Step 1** : Find a godd token which doesn't have a LP pool yet and copy it's CA(Contract Address).
- You can use tools like Apepro, GmGn, Dexscreener, Birdeye to discover tokens.

**Step 2** : Open MetEngine TG bot and select 
<img src="/damm-main.png" alt="A wild meteor" width="500" height="500"></img>

**Step 3** : Select `Open Position` to create a DAMM V2 position.
<img src="/damm-2.png" alt="A wild meteor" width="500" height="500"></img>

**Step 4** : Enter the CA of the token you want to create a position in.
<img src="/damm-3.png" alt="A wild meteor" width="500" height="500"></img>

**Step 5** : If there is no existing pool then you will get an option to create a pool like below.
<img src="/damm-4.png" alt="A wild meteor" width="500" height="500"></img>
- MetEngine will create the pool for you,swap the amount in half and deposit it into the pool alltogether.

**Step 6** : If there is a poll already existing then we will find it for you and you can directly LP into it.
<img src="/damm-5.png" alt="A wild meteor" width="500" height="500"></img>

**Step 7** : You will get a message like this after successful position creation.
<img src="/damm-6.png" alt="A wild meteor" width="500" height="500"></img>

<Callout> You can also check you DAMM V2 position stats on the bot using the `/damm_positions` command.</Callout>
