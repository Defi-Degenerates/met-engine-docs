### Steps to follow to sell the tokens :
    **Step 1** : Use the `/wallet` command to go to your wallet.
    <img src="/wallet.png" width="500" height="500"></img>
    **Step 2** : Select `Manage wallet`. <br/>
    **Step 3** : Select the `Sell Tokens` button.
<img src="/sell-token.png" alt="Sell Tokens" width="500" height="500"></img>
    **Step 4** : Select the % of the token you want to sell (Eg : 100% or 50%).<br/>
    <img src="/select-percentage.png" alt="Select % of token" width="500" height="500"></img>
    **Step 5** : Select the token from the list. <br/>
    <img src="/select-token.png" alt="Select token" width="500" height="500"></img>
    **Step 6** : The selected tokens will be swapped back to `SOL`.
<img src="/sell-success.png" alt="Token Swapped" width="500" height="500"></img>
